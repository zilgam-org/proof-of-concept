SpiralX

SpiralX is a series of art pieces based
on ellipses with varying combinations
of shapes, sizes and colours to generate
a distinct and appealing artwork each time.

Projects

Key Features

Uniquely Generated

Each ellipse based artwork which is generated is unique and is dependent on the transaction which is used to mint the token that it is backed by. No two pieces can thus look alike.

NFT

NFTs are used to tokenize and proove ownership of all the artwork. ZRC-1 tokens consume only a fraction of the energy that ERC-721 and other tokens do, making them more environmentally friendly.

Powered by the Blockchain

The Zilliqa blockchain also immutably stores the scripts that are used to generate the artpieces using the InterPlanetary File System. All transactions are also recorded on this tamper-proof distributed ledger, providing full transparency in next generation auctions.
